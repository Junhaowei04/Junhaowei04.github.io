# MomenT 博客

当前版本先复刻 Butterfly/Hexo 风格博客的主要页面效果：

- 首屏全屏背景图。
- 顶部透明导航，滚动后变成白色导航。
- 居中的站点标题和副标题。
- 底部下滑箭头。
- 滚动后进入文章卡片列表。
- 右侧作者卡、公告、最新文章、分类、标签、归档、网站信息。
- 返回顶部按钮。

这是一个静态前端复刻骨架，不包含真实后端和真实文章详情页。

## 修改内容

作者名是 `MomenT`。当前文章、标签、分类都为空。

文章、标签、分类在 `assets/content.js`。

## 本地预览与上线

`http://127.0.0.1:8787/index.html` 是本机预览地址，只能在当前电脑访问。

要变成随时随地能打开的真实网站，需要部署到公网平台，例如 GitHub Pages、Vercel、Netlify，或自己的云服务器。静态博客不一定需要后端服务器，但一定需要有公网托管服务来提供这些文件。

首页背景图是 `assets/site-cover.png`。

头像是 `assets/avatar.svg`。

主要布局在 `index.html`。

视觉样式在 `assets/styles.css`。

滚动导航、返回顶部和列表渲染在 `assets/app.js`。
